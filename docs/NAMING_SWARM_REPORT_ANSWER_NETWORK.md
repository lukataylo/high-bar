# Naming swarm report — the human answer network ("High Bar")

**Run date:** 2026-07-07
**Brief:** name the expert answer network — vetted human answers sold to people and AI
agents, asked via PWA or MCP/API, escrow-held payment, experts paid on acceptance.
**Method:** same swarm as the Skill Doctor run — 5 independent generators (one lens
each, no shared context) → deterministic frequency + morpheme clustering → 24-name
shortlist → 4 web-research agents → orchestrator scoring.
**Scale:** 129 names generated, 111 unique, 24 researched.

**Over-mined morphemes — this brief DID produce a machine-naming vein:**

- **`-vet-` appeared in 7 names** (anvet, survet, truvet, vetro, vetta, vetted,
  vettra) — the coined-brandable generator strip-mined "vetted." Any vet-compound
  should be treated as the AI-generic fingerprint for this product.
- `-sag-` ×4 (saga, sagax, sagehall, sagen) — the "sage" vein, same problem.
- `line` ×4 (hotline, lifeline, redline; `opine` is a substring false positive and
  was exempted).

The consensus register for this brief is **notary/courtroom English** (vouch, attest,
cosign, gavel, quorum, surety) plus **Greek wise-men** (nestor, solon). All demoted.

## 4. Recommendation

**Cleanest domain path: `seshat`** (only open-source usage anywhere) or **`aval`**
(aval.com looks unclaimed and the word literally means a third-party guarantee —
the escrow story writes itself).

**Richest brand story: `kvasir`** — a being made from the pooled wisdom of many
humans, consulted because he could answer any question. That is this product, in
one myth, and `kvasir` types beautifully as an MCP/API name. **`mensch`** is the
warmer consumer-facing alternative: "your agent hit a wall — ask a mensch."

**Single pick: `kvasir`** for a developer/agent-first brand (with `mensch` the
better pick if the human/consumer side leads). Next step before committing: real
WHOIS + USPTO/EUIPO + App Store + GitHub-org checks on kvasir, mensch, seshat, and
aval — and note kvasir.ai (hedge fund) and Kvasir Analytics would coexist.
